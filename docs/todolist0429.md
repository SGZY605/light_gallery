1. 标签管理页面支持对标签【增、删、改】操作、用户管理页面（管理员）支持【删除】用户账号。
2. 上传照片页面选择照片后显示在队列里，不要立刻上传，支持对队列中的待上传照片进行叉号删除操作，确认无误之后，需要前端有一个上传按钮，点击之后再上传至oss。
3. 删除【首页】这个页面，默认显示【图库】页面，左边导航栏的顺序由上至下依次是：图库、相册、地图、标签、上传、分享、用户、设置。（普通成员没有【用户】管理页面）
3. 点击图片缩略图打开图片的详情页，需要是一个单独的界面铺满全屏，而不是悬浮窗的形式。从缩略图到详情页的图片加一个图片放大的动画：从当前缩略图的位置扩展至详情页图片的位置。 详情页图片后面的背景用这个图片的高斯模糊或者主题颜色当成背景。图片需要支持拖动和滚轮缩放。详情页按键盘esc按键或者右上角的关闭按钮关闭照片详情页。
4. 图片点开详情页，右侧支持显示详细信息：名称、大小、像素、拍摄时间、位置、拍摄参数等等的所有【相机拍摄的】元数据，要求对相机摄影的照片描述友好，参考那种专业的看图工具，没有的留空。此外图片的详情页需要支持修改当前图片的标签和修改地理位置，地理位置需要加一个小的地图组件，可以拖动点击位置，也可以输入经纬度。
